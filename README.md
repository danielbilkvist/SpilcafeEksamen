Spilcaféen projekt til 1. Semester Eksammen

Min kodede forbedring af IXD projektet: Spilcaféen.
Jeg har lavet ændringer, tilføjelser og semantisk struktur i både HTML, CSS og JS.

Jeg har lavet en intern JSON fil for at kunne uploade mine egne spil, og ekspirimentere med åbbygningen af det, den indeholder også alt den data der var i den originale.

Der er kun en HTML fil, som er hoved siden, da jeg har valgt kun at fokusere på selve spiloversigts siden.
